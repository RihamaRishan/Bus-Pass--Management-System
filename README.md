 Bus Pass Management System

This is a web-based Bus Pass Management System developed as an academic project.

 Project Overview
The system is designed to manage bus pass-related processes with proper navigation between modules.

 Features
- User interface with functional navigation
- Bus pass application pages
- Admin and user modules (navigation level)
- Structured page flow

 Technologies Used
- HTML
- CSS
- JavaScript
- PHP 

 Note
This project was developed for learning purposes. Database connectivity is not fully implemented, but the navigation and system flow are functional.
